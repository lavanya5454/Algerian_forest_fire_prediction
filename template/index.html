About Fire Weather Index (FWI)

The Fire Weather Index is a comprehensive system for rating fire danger based on weather conditions

Input Components:
  • FFMC: Fine Fuel Moisture Code
  • DMC: Duff Moisture Code
  • DC: Drought Code
  • ISI: Initial Spread Index
  • BUI: Buildup Index
Risk Levels:
Very Low (0-1)
Low (1-4)
Moderate (4-10)
High (10-17)
Very High (17+)
